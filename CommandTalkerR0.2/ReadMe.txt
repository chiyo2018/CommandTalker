Minecraft Plugin
This is a Minecraft plugin designed for PaperMC servers, offering integration with OpenAI.

-Requirements
Server Software: PaperMC 1.21.3
Plugin Version: English version
API Key: You need to provide your own OpenAI API key in the config.yml file for the plugin to work.

-Setup
Download the plugin and place it in the plugins folder of your PaperMC server.
Start the server to generate the default configuration file (config.yml).
Open the config.yml file and enter your OpenAI API key in the designated field.
Restart the server to apply the changes.

-Configuration (config.yml)
Make sure to add your OpenAI API key in the config.yml file under the appropriate section.

openai-api-key: "your-api-key-here"

-Features
This plugin integrates with OpenAI's API to provide various functionalities within Minecraft.
Fully compatible with PaperMC 1.21.3.
English version of the plugin.
